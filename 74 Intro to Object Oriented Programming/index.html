Object Oriented Programming

Object-oriented programming (OOP) in JavaScript can help developers write more maintainable, scalable, and efficient code for complex applications
In JavaScript, OOP is implemented using prototypal inheritance, which involves linking an object to a prototype. Prototypes contain all methods that are accessible to all objects linked to the prototype
